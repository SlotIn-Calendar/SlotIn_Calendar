# **🗓 SlotIn**

### **일정이 자주 변하는 사람을 위한 시간 슬롯 기반 캘린더**

> 빈 시간을 한눈에, 하루를 더 효율적으로
> 

SlotIn은 일정이 수시로 바뀌는 크리에이터, 프리랜서, 프로젝트 기반 작업자를 위해 

**시간 슬롯 단위로 설계된 iOS 캘린더 서비스**입니다.

---

### Problem

기존 캘린더는 “정해진 일정” 관리에는 적합하지만, 다음과 같은 사용자에게는 오히려 불편합니다.

- 일정이 **자주 변경**된다
- 하루에 **여러 일정이 겹친다**
- 실제로 **언제 시간이 비어 있는지** 한눈에 보기 어렵다
- 일정 수정 시 터치 횟수와 맥락 전환 비용이 크다

---

### **🎯 Solution**

SlotIn은 문제를 이렇게 해결합니다:

✅ **시간 슬롯 중심 UI**

→ 실제 시간 흐름과 동일한 구조의 그리드

✅ **드래그 & 롱프레스 기반 편집**

→ 일정 생성·이동·시간 조정을 한 동작으로

✅ **겹침 일정 시각화**

→ 겹치는 일정도 명확히 구분되는 레이아웃

---

### **✨ Key Features**

| **주요 기능** | **설명** |
| --- | --- |
| 주 / 일 / 월 뷰 | 상황에 맞게 전환 가능한 캘린더 |
| Apple Calendar 연동 | EventKit 기반 기존 일정 불러오기 |
| 시간 슬롯 그리드 | 실제 시간 흐름 기반 시각화 |
| Long Press 일정 확인 | 빈 시간대를 길게 눌러 확인 |
| 실시간 현재 시간 표시 | 현재 시각 타임라인 강조 |
